# Lab 3: Advanced Dart Practice Exercises

## Objectives

Củng cố hiểu biết về các tính năng Dart nâng cao — bao gồm lập trình bất đồng bộ, streams, microtasks, xử lý JSON và factory constructors — thông qua 5 bài tập thực hành.

> [!info] Công cụ thực hành
> Tất cả bài tập có thể chạy trực tiếp trên **DartPad** hoặc **Android Studio / VS Code**.

## Guided Steps

### Exercise 1 — Product Model & Repository

**Goal:** Hiểu Futures và Streams.

1. Định nghĩa `Product { id, name, price }`.
2. Implement `ProductRepository` với:
   - `Future<List<Product>> getAll()`
   - `Stream<Product> liveAdded()` cho real-time updates
3. Dùng `StreamController.broadcast()` để emit items mới.
4. In kết quả ra console.

---

### Exercise 2 — User Repository with JSON

**Goal:** Thực hành JSON serialization / deserialization.

1. Tạo `User { name, email }` và constructor `User.fromJson(Map)`.
2. Giả lập JSON list từ một API.
3. Dùng `Future<List<User>>` để trả về dữ liệu đã parse.
4. Hiển thị kết quả bằng `print()`.

---

### Exercise 3 — Async + Microtask Debugging

**Goal:** Phân biệt microtask queue và event queue.

1. Viết đoạn code với `scheduleMicrotask()` và `Future(() {...})`.
2. In thứ tự thực thi.
3. Giải thích tại sao microtasks chạy trước event callbacks.

> [!question] Câu hỏi tư duy
> Tại sao `scheduleMicrotask` lại được ưu tiên hơn `Future`? Điều này ảnh hưởng như thế nào đến hiệu năng?

---

### Exercise 4 — Stream Transformation

**Goal:** Dùng functional stream operators.

1. Tạo một stream các số từ 1-5.
2. Transform các giá trị thành bình phương bằng `map()`.
3. Lọc số chẵn bằng `where()`.
4. Listen và in mỗi giá trị được emit.

---

### Exercise 5 — Factory Constructors & Cache

**Goal:** Minh họa cách factory constructors implement caching.

1. Tạo class `Settings` với private constructor.
2. Thêm `factory Settings()` trả về singleton instance.
3. Xác minh hai instances tham chiếu đến cùng một object (`identical(a, b) → true`).
