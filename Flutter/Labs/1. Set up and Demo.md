# Lab 1: Flutter Environment Setup & First App
***
## 1. Objective
This lab focuses on setting up the Flutter development environment and building your first application. You will learn to install necessary tools, create a project, execute it on an emulator/device, and perform basic UI customization.

## 3. Guided Exercises

### Exercise 1: Flutter Environment Setup
**Goal:** Install and verify Flutter tools.

- **Steps:**
  1. Install Flutter SDK.
  2. Add Flutter to your system `PATH`.
  3. Run the following command in your terminal:
     ```bash
     flutter doctor
     ```
  4. Ensure all items are marked as properly configured.
  5. Verify that the **Flutter** and **Dart** plugins are installed in Android Studio.

- **Deliverables:**
  - Screenshot of `flutter doctor` output.
  - Screenshot of the Flutter/Dart plugin configuration screen.

### Exercise 2: Create and Run Your First App
**Goal:** Create a project and run the default counter app.

- **Steps:**
  1. Open Android Studio and select **New Flutter Project**.
  2. Name the project: `hello_flutter_lab1`.
  3. Explore the project structure: `lib/`, `android/`, `ios/`, and `pubspec.yaml`.
  4. Start an emulator or connect a physical device.
  5. Run the default counter app.
  6. Modify the `AppBar` title in `main.dart`:
     ```dart
     title: 'My First Flutter App',
     ```
  7. Use **Hot Reload** to observe the changes.

- **Deliverables:**
  - Screenshot of the project structure.
  - Screenshot of the running counter app.
  - Screenshot showing the updated title after Hot Reload.

### Exercise 3: Customize Your First Flutter UI
**Goal:** Build a custom UI using standard widgets.

- **Steps:**
  1. Replace the code in `main.dart` with the following:
     ```dart
     import 'package:flutter/material.dart';

     void main() => runApp(const MyApp());

     class MyApp extends StatelessWidget {
       const MyApp({super.key});

       @override
       Widget build(BuildContext context) {
         return MaterialApp(
           home: Scaffold(
             appBar: AppBar(
               title: const Text('Lab 1 Demo'),
               backgroundColor: Colors.deepPurple,
             ),
             body: const Center(
               child: Column(
                 mainAxisAlignment: MainAxisAlignment.center,
                 children: [
                   Icon(Icons.flutter_dash, size: 80, color: Colors.deepPurple),
                   SizedBox(height: 20),
                   Text('Welcome to Flutter!', style: TextStyle(fontSize: 26, fontWeight: FontWeight.bold)),
                   SizedBox(height: 10),
                   Text('Your first customized layout 😊', style: TextStyle(fontSize: 18, color: Colors.black54)),
                 ],
               ),
             ),
             floatingActionButton: FloatingActionButton(
               onPressed: null,
               backgroundColor: Colors.deepPurple,
               child: const Icon(Icons.add),
             ),
           ),
         );
       }
     }
     ```
  2. Perform a **Hot Reload** to verify the UI updates.
  3. Experiment by modifying colors and text, then reload again.
