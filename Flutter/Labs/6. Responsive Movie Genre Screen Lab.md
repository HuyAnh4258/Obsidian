# Lab 6: Building a Responsive Movie Genre Browsing Screen

## Objectives

Thiết kế và implement một **responsive movie browsing screen** tự động thích ứng với các kích thước màn hình khác nhau (điện thoại nhỏ, lớn, tablet và web).

> [!info] Module liên quan
> Lab này củng cố các kiến thức từ **[[6. Responsive UI & Adaptive Layouts]]**, bao gồm:
> - Dùng **MediaQuery** để đọc kích thước màn hình
> - Dùng **LayoutBuilder** để thích ứng với không gian có sẵn
> - Dùng **Wrap**, **Expanded**, và **ListView/GridView** để build responsive layouts

## Requirements Checklist

**Lab 6.1 — Responsive Hero & Heading Section:**
- [ ] Hiển thị title heading "Find a Movie"

**Lab 6.2 — Search, Genre Chips & Sort Bar:**
- [ ] Search bar cho phép nhập tên phim
- [ ] Set of genre chips (Action, Drama, Comedy...) có thể toggle on/off
- [ ] Sort dropdown với các tuỳ chọn: A-Z, Z-A, Year, Rating

**Lab 6.3 — Responsive Movie List & Tablet Layout:**
- [ ] Hiển thị movie cards với poster, title, year
- [ ] Lọc theo search text (case-insensitive)
- [ ] Lọc theo selected genres
- [ ] Phone (< 800px): single column list
- [ ] Tablet/Wide (≥ 800px): two-column GridView

**Technical:**
- [ ] Chỉ dùng Flutter + Dart (không có third-party packages)

## Guided Steps

### Step 1 — Project Setup

```bash
flutter create lab6_responsive_ui
```
Hoặc mở **DartPad → New Flutter**.

---

### Step 2 — Define Movie Model & Sample Data

```dart
class Movie {
  final String title;
  final int year;
  final List<String> genres;
  final String posterUrl;
  final double rating;
}
```

Định nghĩa constant list `allMovies` với 3-6 items.

---

### Step 3 — Build the Base Scaffold

1. Tạo widget `ResponsiveMovieApp` trả về `MaterialApp`.
2. Set home thành `GenreScreen` (stateful widget).
3. Trong `GenreScreen`, wrap content bằng `SafeArea` và `Padding`.

Layout cơ bản:
```
Title: "Find a Movie"
↓
Search bar
↓
Genre chips (Wrap)
↓
Sort dropdown
↓
Movie list (Expanded)
```

---

### Step 4 — Implement Responsive Search Bar

1. Thêm `TextField` bọc trong rounded container.
2. Thêm state variable `String searchQuery = '';`.
3. Cập nhật `searchQuery` trong `onChanged` bằng `setState`.

---

### Step 5 — Implement Genre Chips using Wrap

```dart
Wrap(
  spacing: 8,
  children: genres.map((genre) => FilterChip(
    label: Text(genre),
    selected: selectedGenres.contains(genre),
    onSelected: (selected) => setState(() {
      selected ? selectedGenres.add(genre) : selectedGenres.remove(genre);
    }),
  )).toList(),
)
```

---

### Step 6 — Implement Sort Dropdown

1. Tạo string values cho sort options (A-Z, Z-A, Year, Rating).
2. Thêm `DropdownButton<String>` gắn với state variable `selectedSort`.
3. Cập nhật sort variable và gọi `setState` trong `onChanged`.

---

### Step 7 — Filter and Sort Movie List

```dart
List<Movie> get visibleMovies {
  var result = allMovies.where((m) {
    final matchSearch = m.title.toLowerCase().contains(searchQuery.toLowerCase());
    final matchGenre = selectedGenres.isEmpty ||
        m.genres.any((g) => selectedGenres.contains(g));
    return matchSearch && matchGenre;
  }).toList();

  // Apply sort...
  return result;
}
```

---

### Step 8 — Build Responsive Movie List

```dart
LayoutBuilder(
  builder: (context, constraints) {
    if (constraints.maxWidth < 800) {
      return ListView.builder(...); // Single column
    } else {
      return GridView.count(crossAxisCount: 2, ...); // Two columns
    }
  },
)
```
