# Lab 8: Building an API-powered List Screen

## Objectives

Xây dựng một Flutter screen lấy dữ liệu từ public REST API, parse JSON thành model classes và hiển thị data bằng `ListView`.

> [!info] Module liên quan
> Lab này củng cố các kiến thức từ **[[8. Working with RESTful APIs & JSON]]**, bao gồm:
> - Thực hiện network requests với `http` package
> - Decode JSON responses
> - Chuyển đổi JSON thành Dart model classes
> - Hiển thị async data dùng `FutureBuilder`
> - Xử lý loading và error states
> - (Optional) Gửi data qua POST requests

### Step 2 — Call a Public API (Lab 8.1)

```dart
import 'package:http/http.dart' as http;
import 'dart:convert';

Future<void> fetchPosts() async {
  final response = await http.get(
    Uri.parse('https://jsonplaceholder.typicode.com/posts'),
  );
  
  if (response.statusCode == 200) {
    print(response.body); // In raw JSON để kiểm tra
  } else {
    throw Exception('Failed to load posts');
  }
}
```

**Goal:** Lấy được API data thành công.

---

### Step 3 — Convert JSON → Model (Lab 8.2)

```dart
class Post {
  final int id;
  final String title;
  final String body;
  
  Post({required this.id, required this.title, required this.body});
  
  factory Post.fromJson(Map<String, dynamic> json) {
    return Post(
      id: json['id'],
      title: json['title'],
      body: json['body'],
    );
  }
}

// Parse list:
List<Post> parsePosts(String responseBody) {
  final parsed = json.decode(responseBody) as List;
  return parsed.map((p) => Post.fromJson(p)).toList();
}
```

---

### Step 4 — Display Data Using FutureBuilder (Lab 8.2 & 8.4)

```dart
FutureBuilder<List<Post>>(
  future: apiService.getPosts(),
  builder: (context, snapshot) {
    if (snapshot.connectionState == ConnectionState.waiting) {
      return const CircularProgressIndicator();
    } else if (snapshot.hasError) {
      return Text('Lỗi: ${snapshot.error}');
    } else {
      return ListView.builder(
        itemCount: snapshot.data!.length,
        itemBuilder: (context, index) {
          final post = snapshot.data![index];
          return ListTile(
            leading: Text('#${post.id}'),
            title: Text(post.title),
          );
        },
      );
    }
  },
)
```

> [!tip] FutureBuilder states
> - `ConnectionState.waiting` → Đang loading
> - `snapshot.hasError` → Có lỗi
> - `snapshot.hasData` → Có dữ liệu để hiển thị

### Step 6 — Organize Code into ApiService (Lab 8.4)

```dart
class ApiService {
  final http.Client client;
  
  ApiService({http.Client? client}) : client = client ?? http.Client();
  
  Future<List<Post>> getPosts() async {
    final response = await client.get(
      Uri.parse('https://jsonplaceholder.typicode.com/posts'),
    );
    if (response.statusCode == 200) {
      return parsePosts(response.body);
    }
    throw Exception('Failed to load posts: ${response.statusCode}');
  }
}
```

**Goal:** Tách biệt networking logic khỏi UI (Separation of Concerns).

---

### Step 7 — Optional POST Request

```dart
Future<Post> createPost(String title, String body) async {
  final response = await http.post(
    Uri.parse('https://jsonplaceholder.typicode.com/posts'),
    headers: {'Content-Type': 'application/json'},
    body: json.encode({'title': title, 'body': body, 'userId': 1}),
  );
  return Post.fromJson(json.decode(response.body));
}
```
