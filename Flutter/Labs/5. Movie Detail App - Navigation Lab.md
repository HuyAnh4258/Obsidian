# Lab 5: Building A Movie Detail App With Navigation

## Objectives

Thiết kế và implement một **Movie Detail App** đơn giản, minh họa cách dùng navigation mechanisms của Flutter (`Navigator.push`, `MaterialPageRoute`, hoặc named routes) và UI composition cơ bản.

> [!info] Module liên quan
> Lab này củng cố các kiến thức từ **[[5. Navigation & State Management]]**.

## Requirements Checklist

**Home Screen:**
- [ ] Danh sách movies cuộn được (poster, title, rating)
- [ ] Điều hướng đến Movie Detail khi tap

**Movie Detail Screen:**
- [ ] Poster (Hero banner với gradient)
- [ ] Title và genres dạng chips
- [ ] Overview text
- [ ] Action buttons (Favorite / Rate / Share)
- [ ] Danh sách trailers

**Technical:**
- [ ] Navigation dùng `Navigator.push` + `MaterialPageRoute` **hoặc** Named routes
- [ ] Truyền `Movie` object giữa các screens
- [ ] Layout cuộn được và responsive
- [ ] Dùng static sample data (không cần API)

**Optional Enhancements:**
- [ ] Implement Favorite toggle với state update
- [ ] Thêm search bar lọc movies
- [ ] (Nâng cao) Deep-link mở trực tiếp movie detail page

---

## Guided Steps

### Step 1 — Project Setup

```bash
flutter create movie_app
```
Hoặc mở DartPad Flutter.

---

### Step 2 — Define Data Model

Tạo class `Movie` với các fields:

```dart
class Movie {
  final int id;
  final String title;
  final String posterUrl;
  final String overview;
  final List<String> genres;
  final double rating;
  final List<String> trailers;
}
```

Thêm file `sample_data.dart` chứa 2-3 movie objects.

---

### Step 3 — Build Home Screen

- Dùng `ListView.builder` để hiển thị mỗi movie card.
- Khi tap → navigate đến Movie Detail screen, truyền selected object.

---

### Step 4 — Build Movie Detail Screen

Xây dựng UI từng bước:

1. `Scaffold` + `AppBar` trống
2. Hero Banner (`Stack` + `Image.network` + gradient)
3. Title & Genres (`Column` + `Wrap` + `Chip`)
4. Overview text với `Padding`
5. Row of `IconButton` (Favorite, Rate, Share)
6. Trailer list dùng `ListView.builder`

---

### Step 5 — Test and Polish

- Chạy app → kiểm tra navigation hoạt động và UI cuộn mượt.
- (Optional) Thêm state management với `setState()` hoặc `ChangeNotifier`.

> [!warning] Lưu ý Navigation
> Khi dùng `Navigator.push`, đảm bảo bạn truyền đúng `BuildContext`. Tránh dùng context sau khi widget đã bị disposed.

---

## Expected Results

App hoàn chỉnh phải:
- Hiển thị movie list ở Home.
- Điều hướng đến detail page với poster, genres, overview, actions và trailers.
- Quay về Home bằng nút Back.
