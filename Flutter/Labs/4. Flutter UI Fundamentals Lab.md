# Lab 4: Flutter UI Fundamentals

## Objectives

Thực hành xây dựng giao diện người dùng bằng hệ thống widget cốt lõi của Flutter. Học cách làm việc với basic widgets, input controls, layout structures, navigation scaffolding và các kỹ thuật polish như spacing và theming.

> [!info] Module liên quan
> Lab này tương ứng trực tiếp với **[[4. Flutter UI Fundamentals]]**.

## Requirements Checklist

- [ ] Exercise 1: Core Widgets (Text, Image, Icon, Card, ListTile)
- [ ] Exercise 2: Input Widgets (Slider, Switch, RadioListTile, Pickers)
- [ ] Exercise 3: Layout Composition (Column, Row, Padding, ListView)

## Guided Steps

### Exercise 1 — Core Widgets: Text, Image, Icon, Card, ListTile

**Goal:** Xây dựng UI đơn giản với các display widget thiết yếu của Flutter.

1. Tạo file mới `core_widgets_demo.dart`.
2. Build một screen có:
   - Một `Text` dạng headline
   - Một `Icon` dùng Material Icons
   - Một `Image.network()` (bất kỳ URL hình ảnh hợp lệ)
   - Một `Card` chứa `ListTile`
3. Chạy UI và xác nhận các widget hiển thị đúng.

> [!tip] Widget cần nhớ
> `Text`, `Icon`, `Image.network`, `Card`, `ListTile`

---

### Exercise 2 — Input Widgets: Slider, Switch, RadioListTile, DatePicker

**Goal:** Xây dựng interactive UI cho phép người dùng điều khiển giá trị.

1. Tạo `StatefulWidget` tên `InputControlsDemo`.
2. Implement `Slider`, `Switch`, và `RadioListTile` group.
3. Thêm button hiển thị `DatePicker` khi nhấn.
4. Hiển thị các giá trị đã cập nhật lên màn hình.

> [!warning] Lưu ý
> Input widgets cần `StatefulWidget` để quản lý trạng thái. Đảm bảo gọi `setState()` khi giá trị thay đổi.

---

### Exercise 3 — Layout Basics: Column, Row, Padding, ListView

**Goal:** Xây dựng sectioned UI layout tương tự Home screen của app thực tế.

1. Dùng `Column` để tạo vertical sections.
2. Thêm spacing bằng `Padding` và `SizedBox`.
3. Thêm `ListView.builder` hiển thị danh sách movie titles hoặc items.
4. Áp dụng consistent spacing (8, 12, hoặc 16 px).

---

### Exercise 4 — App Structure with Scaffold, AppBar, FAB & Theme

**Goal:** Thực hành xây dựng complete screen structure.

1. Tạo màn hình mới dùng `Scaffold`.
2. Thêm:
   - `AppBar`
   - `Body`
   - `FloatingActionButton`
   - Tùy chỉnh Theme bằng `ThemeData`
3. Implement toggle "Dark Mode" dùng `themeMode`.

---

### Exercise 5 — Debug & Fix Common UI Errors

**Goal:** Hiểu các lỗi layout phổ biến và cách fix.

| Vấn đề | Giải pháp |
| :--- | :--- |
| `ListView` bên trong `Column` bị overflow | Wrap ListView bằng `Expanded` |
| Overflow trên màn hình nhỏ | Dùng `SingleChildScrollView` |
| State không update | Thêm `setState()` |
| DatePicker lỗi BuildContext | Gọi từ đúng vị trí trong widget tree |
