> [!note]+ # Conversion Functions
> > [!note]+ ### CAST Function
> > - Converts an expression of one data type to another
> > 
> > ```sql
> > CAST ( expression AS data_type [ ( length ) ] )
> > ```
> > [!note]+ ### CONVERT Function
> > > When you convert expressions from one type to another, in many cases there will be a need within a stored procedure or other routine to convert data from a datetime type to a varchar type.
> > 
> > ```sql
> > CONVERT ( data_type [ ( length ) ] , expression [ , style ] )
> > ```
> [!note]+ # Date and Time Functions
> > [!note]+ ### GETDATE() & DATEPART() Function
> > - The GETDATE() function returns the current date and time from the SQL Server.
> > - The DATEPART() function is used to return a single part of a date/time, such as year, month, day, hour, minute, etc.
> > 
> > ```sql
> > GETDATE()
> > DATEPART(datepart, date) --YYYY, MMMM, DDDD...
> > ```
> > [!note]+ ### DAY, MONTH,YEAR Function
> > - Returns an integer representing the day/month/year (day of the month) of the specified date.
> > 
> > ```sql
> > SELECT DAY(GETDATE()) AS [Day],
> > MONTH(GETDATE()) AS [Month],
> > YEAR(GETDATE()) AS [Year]
> > ```
> > [!note]+ ### DATEADD Function
> > - The DATEADD() function adds or subtracts a specified time interval from a date.
> > 
> > ```sql
> > DECLARE @dt datetime;
> > SET @dt = GETDATE();
> > SELECT @dt AS CurrentDate;
> > SELECT DATEADD(day, 30, @dt) AS AfterDate;
> > ```
> > [!note]+ ### DATEDIFF Function
> > - The DATEDIFF() function returns the time between two dates.
> > 
> > ```sql
> > DECLARE @date1 DATETIME;
> > DECLARE @date2 DATETIME;
> > SET @date1= '2012-04-07 20:12:22.013‘;
> > SET @date2= '2014-02-27 22:14:10.013‘;
> > SELECT DATEDIFF(month, @date1, @date2) AS 'Month'
> > ```
> [!note]+ # String Functions
> > [!note]+ ### RTRIM, LTRIM Function
> > - LTRIM Removes all white spaces on the left of the string.
> > - RTRIM Removes all white spaces on the right of the string.
> > 
> > ```sql
> > SELECT LTRIM(' Sample '); -- 'Sample '
> > SELECT RTRIM(' Sample '); -- ' Sample'
> > ```
> > [!note]+ ### SUBSTRING Function
> > - The Substring function in SQL is used to return a portion of string. This
> > function is called differently in different databases:
> > 
> > ```sql
> > SUBSTRING(str, position, length)
> > --Example
> > SELECT SUBSTRING('Bill Gates', 0 ,5) As Result
> > ```
> > [!note]+ ### LEN, CHARINDEX, PATINDEX Function
> > - The CHARINDEX and PATINDEX functions return the starting position of a pattern you specify.
> > - PATINDEX can use wildcard characters, but CHARINDEX cannot
> > 
> > ```sql
> > LEN(str)
> > CHARINDEX ( expression1 ,expression2 [ , start_location ] )
> > PATINDEX ( '%pattern%' , expression )
> > 
> > --Example
> > SELECT CHARINDEX('bicycle', 'Reflectors of your bicycle.')
> > AS Positions
> > SELECT PATINDEX ( '%ein%', 'Das ist ein Test') AS Positions
> > ```
