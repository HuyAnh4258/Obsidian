> [!note]+ # Tổng quan về `Lambda`
> > [!note]+ ### Định nghĩa
> > > `Biểu thức Lambda` là một khối code được thiết kế để nhận vào một tập tham số truyền vào và trả về giá trị nào đó.
> > 
> > - `Lambda` giống như `phương thức vô danh` vì nó không cần đặt tên, có thể được triển khai ngay trong body của một phương thức khác.
> > - `Biểu thức Lambda` thường được gọi ngắn gọn là `Lambda`
> > - `Lambda` được dùng để triển khai một phương thức/ hàm trừu tượng duy nhất bên trong `functional interface` (được thêm vào từ Java 8) với các chức năng sau:
> >     - Có thể coi một chức năng như một đối số, hoặc coi một đoạn code là data
> >     - Một hàm có thể được tạo ra mà không thuộc về bất cứ Class nào
> >     - Một biểu thức lambda có thể được truyền qua lại như một đối tượng và khởi chạy bất kỳ đâu gọi tới
> > [!note]+ ### Cú pháp
> > ```java
> > (parameter_list) -> { function_body }
> > ```
> > Một biểu thức lambda yêu cầu phải có 3 phần như sau:
> > 1. parameter_list: (có thể rỗng) Nếu như có nhiều hơn một đối số, chúng sẽ ngăn cách bằng dấu `,` và toàn bộ nằm trong dấu `()`
> > be comma separated and enclosed inside the parenthesis.
> > 2. arrow operator: Toán tử mũi tên `->`
> > 3. function_body: là phần thân hàm của lambda, bao gồm các câu lệnh để thực thi (hoặc có thể rỗng). Nếu như thân hàm này chỉ có đúng một dòng thì không cần thiết đóng `{}`
> > [!note]+ ### Các loại Lambda khác tham số
> > Có 3 loại tham số truyền vào lambda như sau:
> > 1. Biểu thức Lambda không có tham số
> > ```java
> > () -> System.out.println("Zero parameter lambda");
> > ```
> > 2. Biểu thức Lambda có 1 tham số
> > ```java
> > (p) -> System.out.println("One parameter: " + p);
> > ```
> > 3. Biểu thức Lambda có nhiều tham số
> > ```java
> > (p1, p2) -> {
> > 	Double total = p1 + p1;
> > 	System.out.println("Multiple parameters: " + p1 + ", " + p2 + ":" + total);
> > };
> > ```
> [!note]+ # Tổng quan về `Functional Interface`
> > [!note]+ ### Định nghĩa
> > > Một `functional interface` là một interface chỉ bao gồm `một phương thức`
> > 
> > - Syntax:
> > 
> > ```java
> > @FunctionalInterface
> > public interface Check {
> > 	boolean test(Candidate c);
> > }
> > ```
> > Về `@FunctionalInterface` annotation
> > - Không bắt buộc để cho code chạy được, chỉ giúp người khác làm rõ hơn về mục đích
> > - Java sẽ luôn coi bất cứ interface chỉ có một phuong thức nào là một functional interface
> > - Nếu như một interface có annotation này mà có hơn 1 phương thức, Java sẽ báo lỗi.
> > [!note]+ ### Áp dụng đối với Lambda
> > > [!note]+ VD 1: Cách dùng biểu thức Lambda cơ bản
> > > ```java
> > > public interface FuncInterface {
> > > 	// An abstract function
> > > 	void abstractFun(int x);
> > > 	
> > > 	// A non-abstract (or default) function
> > > 	default void normalFun() {
> > > 		System.out.println("Hello");
> > > 	}
> > > }
> > > ```
> > > ```java
> > > public interface Test {
> > > 	public static void main(String args[]) {
> > > 	// lambda expression to implement above functional interface.
> > > 	// This interface by default implements abstractFun()
> > > 	FuncInterface fobj = (int x) -> System.out.println(2 * x);
> > > 	// This calls above lambda expression and prints 10.
> > > 	fobj.abstractFun(5);
> > > 	fobj.normalFun();
> > > 	}
> > > }
> > > ```
> > 
> > > [!note]+ VD 2: Sử dụng lambda với hàm `forEach()` để in ra tất cả phần tử trong `List`
> > > ```java
> > > ArrayList<Integer> numbers = new ArrayList<Integer>();
> > > numbers.add(5);
> > > numbers.add(9);
> > > numbers.add(8);
> > > numbers.add(1);
> > > 
> > > numbers.forEach((n) -> {
> > > 	System.out.println(n);
> > > });
> > > ```
> > 
> > > [!note]+ VD 3: Sử dụng interface `Consumer` để lưu một biểu thức lambda như một biến số
> > > ```java
> > > ArrayList<Integer> numbers = new ArrayList<Integer>();
> > > numbers.add(5);
> > > numbers.add(9);
> > > numbers.add(8);
> > > numbers.add(1);
> > > 
> > > Consumer<Integer> method = (n) -> { System.out.println(n); };
> > > numbers.forEach( method );
> > > ```
> > 
> > > [!note]+ VD 4: Tạo ra các phương thức nhận vào biểu thức Lambda
> > > ```java
> > > interface StringFunction {
> > > 	String run(String str);
> > > }
> > > 
> > > public class StringSample {
> > > 	public static void main(String[] args) {
> > > 		StringFunction exclaim = (s) -> s + "!";
> > > 		StringFunction ask = (s) -> s + "?";
> > > 		print("Hello", exclaim);
> > > 		print("Hello", ask);
> > > 	}
> > > 	public static void print(String str, StringFunction format) {
> > > 		String result = format.run(str);
> > > 		System.out.println(result);
> > > 	}
> > > }
> > > ```
