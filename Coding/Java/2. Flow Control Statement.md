> [!note]+ # Arrays
> > [!note]+ ### About Java Arrays
> > > `Array` is a `data structure` that contains `related data items`. Its `size` will be `fixed` and not changable after creation.
> > 
> > - Types
> >     - Single Dimensional Array
> >     - Multidimesional Array
> > ![[da2cc0fc-ac46-4104-97c2-48aa8ea85ab5.png]]
> > [!note]+ ### Single Dimensional Arrays
> > > [!note]+ Declaring an Array
> > > - There are 3 ways to declare an array
> > > 
> > > ```java
> > > datatype[] identifier;
> > > datatype[] identifier = new datatype[size];
> > > datatype[] identifier = {value1,value2,…valueN};
> > > ```
> > > - An alternative way (not recommended)
> > > 
> > > ```java
> > > datatype identifier[];  //this form is discouraged
> > > ```
> > > - Example
> > > 
> > > ```java
> > > byte[] bArray;
> > > float[] fArray = new float[20]; //fixed length from now
> > > int[] iArray = { 32, 27, 64, 18, 95, 14, 90, 70, 60, 37 };
> > > ```
> > 
> > > [!note]+ Array Index
> > > > `Array Index` hay `subscript` là chỉ mục giúp ta truy xuất đến một phần tử trong mảng nằm ở chỉ mục đó. `Index` của một mảng `bắt đầu từ index 0`
> > > 
> > > - Index nằm trong ngoặc vuông `[index]` và đứng sau tên array cần truy xuất đến.
> > > - `Index` bắt đầu từ `0` và luôn `nhỏ hơn độ dài mảng` 1 đơn vị.
> > > 
> > > ```java
> > > int[] iArray = {1, 2, 3, 4, 5} //Array size 5
> > > System.out.println(iArray[0]); //In ra 1
> > > System.out.println(iArray[2]); //In ra 3
> > > ```
> > > - Ta có thể truyền mảng vào trong phương thức để sử dụng mảng đó cho việc tính toán
> > > 
> > > ```java
> > > static int findMax(int[] intArray) {
> > > 	int max = intArray[0];
> > > 	for (int i = 1; i < intArray.length; i++) {
> > > 		intArray[i] *= 2;
> > > 		if (intArray[i] > max) {
> > > 			max = intArray[i];
> > > 		}
> > > 	}
> > > 	return max;
> > > } 
> > > ```
> > > `ArrayOutOfBoundsException` là một `ngoại lệ` (lỗi) thường gặp khi ta thao tác với mảng. Điều này xảy ra khi:
> > > - Mảng được `truy xuất đến` khi `kích cỡ bị âm`.
> > > - Ta chạy vòng `lặp` đến một `index` nằm ngoài các index nằm `trong mảng`
> > > 
> > > ```java
> > > public class TestArrayException {
> > > 	public static void main(String[] args) {
> > > 		int arr[] = { 50, 60, 70, 80 };
> > > 		for (int i = 0; i <= arr.length; i++) {
> > > 			System.out.println(arr[i]);
> > > 		}
> > > 	}
> > > }
> > > ```
> > > 
> > > 
> > [!note]+ ### Multidimesional Arrays (Mảng đa chiều)
> > > `Mảng đa chiều` là một mảng mà `chứa các mảng khác` thành phần tử của nó. Điều đó giúp ta lưu trữ các dạng dữ liệu 2D (dạng bảng), 3D (dạng khối) hoặc nhiều chiều hơn nữa thay vì chỉ là một danh sách tuyến tính.
> > 
> > > [!note]+ Khai báo một mảng đa chiều và tạo ra mảng đa chiều
> > > Để khai báo một mảng đa chiều, ta sẽ thêm `[]` dựa vào số `dimension` mà ta cần cho mảng đó, ví dụ `2D` thì ta có `[][]`, `3D` thì ta có `[][][]`.
> > > ```java
> > > // Declaring and initializing 2D array
> > > int arr[][] = new int[][];
> > > ```
> > 
> > > [!note]+ Khởi tạo mảng đa chiều
> > > Ta có thể khởi tạo giá trị ngay cho mảng đa chiều như sau:
> > > ```java
> > > int arr[][] = { { 1, 2, 3 }, { 2, 4, 5 }, { 4, 4, 5 } };
> > > ```
> > 
> > > [!note]+ In ra mảng đa chiều
> > > Để in ra mảng đa chiều, ta sẽ sử dụng vòng lặp chồng `nested loop` có số tầng bằng số chiều của mảng để chạy qua từng phần tử.
> > > ```java
> > > // Printing 2D array
> > > for (int i = 0; i < 3; i++) {
> > > 	for (int j = 0; j < 3; j++) {
> > > 		System.out.print(arr[i][j] + "\t");
> > > 	}
> > > 	System.out.println();
> > > }
> > > ```
> > [!note]+ ### Jagged Array (Mảng răng cưa)
> > > `Mảng răng cưa` là một `mảng đa chiều không đều` - tức mỗi hàng có đội dài khác nhau.
> > 
> > ```java
> > int[][] jagged = new int[3][]; // 3 hàng, nhưng chưa xác định số cột
> > 
> > jagged[0] = new int[2]; // hàng 1 có 2 phần tử
> > jagged[1] = new int[4]; // hàng 2 có 4 phần tử
> > jagged[2] = new int[3]; // hàng 3 có 3 phần tử
> > ```
> > Khi đó, mảng sẽ được in ra như sau:
> > ```java
> > [0, 0]
> > [0, 0, 0, 0]
> > [0, 0, 0]
> > ```
> > Khi khởi tạo trực tiếp:
> > ```java
> > int[][] jagged = {
> >     {1, 2},
> >     {3, 4, 5, 6},
> >     {7, 8, 9}
> > };
> > ```
> > Điểm mạnh & Điểm yếu:
> > - **Điểm mạnh**: Linh hoạt, tiết kiệm bộ nhớ nếu dữ liệu kiểu bảng có số cột khác nhau.
> > - **Điểm yếu**: Cấu trúc phức tạp, khó để truy xuất vì không thể giả định số cột cố định
> > [!note] 🗣
> > Mảng đa chiều không giới hạn số tầng, nhưng mảng trên 3 chiều rất hiếm khi được dùng vì rất khó đọc và quản lý. Thêm vào đó, hiệu năng và bộ nhớ sẽ giảm mạnh nếu số chiều quá lớn vì ít linh hoạt.
