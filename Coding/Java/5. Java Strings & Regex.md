> [!note]+ # Strings là gì?
> > [!note]+ ### Định nghĩa
> > > `String` hay “chuỗi” là một chuỗi các ký tự `chars` , ví dụ ta có chuỗi “Hello” là một chuỗi bao gồm 5 ký tự.
> > 
> > > [!note] 🗣
> > > Trong Java, String là một `đối tượng bất biến` , nghĩa là nó luôn là một `hằng biến` và `không thể bị thay đổi` một khi đã khởi tạo giá trị
> > 
> > Có 2 cách để khởi tạo String trong Java
> > - Dùng cặp nháy kép `""`
> > - Dùng từ khoá `new` để khởi tạo đối tượng `String`
> > [!note]+ ### Cấp tầng của String
> > > `String` là một đối tượng từ `java.lang.String` được sử dụng để tạo các chuỗi ký tự hoặc là chuỗi các giá trị ký tự
> > 
> > Class `String` `implements` `Serializable` `Comparable` và `CharSequence`
> > - `Serializable` có các phương thức `hỗ trợ read, write` dữ liệu
> > - `Comparable` có các phương thức `so sánh`
> > - `CharSequence` có các phương thức `cơ bản` của chuỗi ký tự
> > 
> > ![[image 11.png]]
> > Một số Class khác như `StringBuffer` và `StringBuilder` cũng triển khai `CharSequence`
> > [!note]+ ### Lớp String
> > - Lớp String có 11 `constructor` và 40+ phương thức
> > - Là một lớp cực kỳ hữu ích khi lập trình
> > 
> > ![[image 12.png]]
> > [!note]+ ### Khởi tạo String
> > Có 2 cách khởi tạo String
> > Sử dụng `string literal` : Java String literal được tạo ra dùng 2 dấu `""`.
> > ```java
> > String s = "Welcome to Java";
> > ```
> > Sử dụng `new` keyword : Khởi tạo thông qua `constructor` bằng 2 cách:
> > - Truyền vào `string literals` 
> > ```java
> > String message = new String("Welcome to Java");
> > ```
> > - Truyền vào một mảng char `char[]` 
> > ```java
> > char[] chars = {'G', 'o', ... }; 
> > String s = new String(chars);
> > ```
> > [!note]+ ### String pool
> > > `Java String pool` là một tập hợp các String được lưu trong bộ nhớ `heap` 
> > Bộ nhớ `heap` này được dùng để chứa các `String literals`
> > 
> > ![[image 13.png|Khi một String object mới được khởi tạo, Java String pool sẽ tìm xem giá trị đó đã tồn tại hay chưa
> > - Nếu đã tồn tại, nó sẽ liên kết đến địa chỉ bộ nhớ đó, khi đó các biến sẽ có chung địa chỉ giá trị
> > - Nếu chưa, một object mới sẽ được tạo ra, pool trả về địa chỉ tương ứng]]
> > Ví dụ:
> > ```java
> > String str1 = "Hello"; 
> > String str2 = "Hello"; 
> > System.out.println(str1 == str2); // Output: true
> > ```
> > Ví dụ 2:
> > ```java
> > String str3 = new String("Hello"); 
> > String str4 = new String("Hello"); 
> > System.out.println(str3 == str4); // Output: false
> > ```
> > Ví dụ 3:
> > ```java
> > String str5 = new String("Hello").intern(); 
> > String str6 = new String("Hello").intern(); 
> > System.out.println(str5 == str6); // Output: true 
> > ```
> > 
> [!note]+ # String bất biến
> > [!note]+ ### Thế nào là bất biến?
> > **String là bất biến**
> > ```java
> > String s = "Java"; //creates a String object
> > s = "HTML"; // creates a new String 
> > ```
> > Giá trị “Java” cũ sẽ vẫn được lưu lại, nhưng nó sẽ không được truy cập đến nữa. Biến s bây giờ sẽ trỏ đến một object mới
> > ![[image 14.png]]
> > > [!note] 🗣
> > > Chính vì việc sử dụng chung các giá trị cho các biến khác nhau nên việc lưu trữ sẽ trở nên hiệu quả và ít tốn bộ nhớ hơn
> > 
> > 
> > [!note]+ ### So sánh String
> > Có 3 kiểu so sánh String
> > - So sánh giá trị: `equals()` và `equalsIgnoreCase()` ứng dụng cho việc xác thực thông tin
> > ```java
> > if (s1.equal(s2))
> >     System.out.println("s1 and s2 are the same content");
> > else
> >     System.out.println(“s1 and s2 are different content");
> > ```
> > - So sánh địa chỉ (reference): `==` dùng để kiểm tra 2 String có cùng trỏ đến một Object
> > ```java
> > if (s1 == s2)
> >     System.out.println("s1 and s2 are the same object");
> > else
> >     System.out.println("s1 and s2 are different objects");
> > ```
> > - Sắp xếp: `compareTo()` sẽ so sánh giá trị của 2 String theo thứ tự bảng chữ cái để quyết định xem String nào “lớn” hơn, và sử dụng thứ tự đó để sắp xếp.
> > [!note]+ ### Bất biến khi làm việc với phương thức
> > Ví dụ
> > ```java
> > public class TestImmutableString {
> > 
> >   public static void main(String[] args) {
> >     String s = "FPT Software";
> >     s.concat(" Academy"); // concat() method appends the string at the end 
> >     System.out.println(s);// will print FPT Software because strings are
> >                           // immutable objects
> >   }
> > }
> > ```
> > ![[image 15.png]]
> > **Giải pháp:** trỏ `s` đến kết quả của `concat()` 
> > ```java
> > s = s.concat(" Academy"); //s trỏ đến giá trị mới
> > ```
> > 
> > > Nhiều phương thức (hàm) của lớp String trong Java không thay đổi chuỗi gốc,
> > mà tạo ra một chuỗi mới chứa phiên bản đã được biến đổi (chuyển đổi, cắt bỏ, chỉnh sửa, v.v).
> > 
> > ```java
> > String sentence = "The quick brown fox jumps over the lazy dog.";
> > String substring = sentence.substring(4, 10); // Creates new string "brown fox" 
> > System.out.println(sentence); // Still contains the complete sentence 
> > System.out.println(substring); // Outputs "quick"
> > ```
> > ```java
> > String sentence = "The quick brown fox jumps over the lazy dog."; 
> > String newSentence = sentence.toUpperCase(); 
> > System.out.println(sentence == newSentence); // Print 'false'
> > ```
> [!note]+ # Các phương thức
> - char `charAt(int index)`: Trả về ký tự tại vị trí (index) được chỉ định trong chuỗi. Nếu index không hợp lệ sẽ bắn ra `IndexOutOfBoundsException`
> ```java
> String str = "Hello";
> char c = str.charAt(1); // 'e'
> System.out.println(c);  // In ra: e
> ```
> - boolean `equals(Object obj)`: So sánh nội dung của chuỗi hiện tại với chuỗi obj. Nếu giống nhau → trả về true, ngược lại → false.
> ```java
> String s1 = "Java";
> String s2 = "Java";
> String s3 = "JAVA";
> 
> System.out.println(s1.equals(s2)); // true
> System.out.println(s1.equals(s3)); // false (phân biệt chữ hoa thường)
> ```
> - int `compareTo(String string)`: So sánh theo thứ tự từ điển (lexicographical order) dựa trên giá trị Unicode của từng ký tự.
> ```java
> System.out.println("abc".compareTo("abc")); // 0
> System.out.println("abc".compareTo("abd")); // -1
> System.out.println("abd".compareTo("abc")); // +1
> ```
> - boolean `startsWith(String prefix)`: Kiểm tra xem chuỗi có **bắt đầu bằng** một chuỗi con `prefix` không.
> ```java
> String s = "Hello World";
> System.out.println(s.startsWith("Hello")); // true
> System.out.println(s.startsWith("World")); // false
> ```
> - boolean `endsWith(String suffix)`: Kiểm tra xem chuỗi có **kết thúc bằng** chuỗi con `suffix` không.
> ```java
> String s = "Hello.java";
> System.out.println(s.endsWith(".java")); // true
> System.out.println(s.endsWith(".txt"));  // false
> ```
> - int `hashCode()`: Trả về một giá trị số nguyên (hash code) đại diện cho nội dung của chuỗi.
> Hash code này thường được dùng trong các cấu trúc dữ liệu như HashMap, HashSet, v.v.
> ```java
> String s = "Java";
> System.out.println(s.hashCode()); // ví dụ: 2301506
> ```
> - int `indexOf(int ch)`: Trả về vị trí đầu tiên của ký tự ch trong chuỗi.
> Nếu không tìm thấy → trả về -1.
> ```java
> String s = "banana";
> System.out.println(s.indexOf('a')); // 1
> System.out.println(s.indexOf('z')); // -1
> ```
> - boolean `contains(CharSequence s)`: Kiểm tra xem chuỗi có chứa chuỗi con `s` hay không. Nếu s truyền vào là `null` sẽ bắn ra `NullPointerException`
> ```java
> String s = "The quick brown fox";
> System.out.println(s.contains("quick")); // true
> System.out.println(s.contains("slow"));  // false
> ```
> - String `concat(String str)`: Nối (ghép) chuỗi `str` vào **cuối** chuỗi hiện tại.
> ```java
> String s1 = "Hello";
> String s2 = s1.concat(" World");
> System.out.println(s2); // "Hello World"
> ```
> - String `trim()`: Xóa **khoảng trắng ở đầu và cuối** chuỗi (nhưng **không** xóa khoảng trắng ở giữa)
> ```java
> String s = "   Java Programming   ";
> System.out.println(s.trim()); // "Java Programming"
> ```
> - byte[] `getBytes()`: Chuyển chuỗi thành mảng byte, sử dụng bộ mã ký tự mặc định (default charset). Thường dùng khi làm việc với file, mạng, hoặc mã hóa dữ liệu.
> ```java
> String s = "ABC";
> byte[] bytes = s.getBytes();
> System.out.println(Arrays.toString(bytes)); 
> // [65, 66, 67] → (mã ASCII của 'A', 'B', 'C')
> ```
> - int `length()`: Trả về độ dài (số ký tự) của chuỗi.
> ```java
> String s = "Hello";
> System.out.println(s.length()); // 5
> ```
> - boolean `matches(String regex)`: Kiểm tra xem chuỗi có phù hợp (match) với biểu thức chính quy (regular expression) hay không.
> ```java
> String s = "abc123";
> System.out.println(s.matches("[a-z]+\\d+")); // true
> System.out.println(s.matches("\\d+"));       // false
> ```
> - static String `valueOf()`: Chuyển **giá trị của biến bất kỳ** (int, float, char, boolean, object, v.v.) sang **chuỗi (**`**String**`**)**.
> ```java
> int n = 123;
> double d = 45.6;
> char c = 'A';
> System.out.println(String.valueOf(n)); // "123"
> System.out.println(String.valueOf(d)); // "45.6"
> System.out.println(String.valueOf(c)); // "A"
> ```
> - char[] `toCharArray()`: Chuyển chuỗi thành **mảng ký tự (**`**char[]**`**)**.
> ```java
> String s = "Java";
> char[] arr = s.toCharArray();
> System.out.println(Arrays.toString(arr)); // [J, a, v, a]
> ```
> - String[] `split(String regex)`: Cắt chuỗi thành **mảng các chuỗi con** dựa trên **biểu thức chính quy (regex)** làm dấu phân cách.
> ```java
> String s = "apple,banana,orange";
> String[] parts = s.split(",");
> System.out.println(Arrays.toString(parts));
> // [apple, banana, orange]
> ```
> - String `format(format, args…)`: Tạo một chuỗi mới dựa trên mẫu định dạng (format) và các giá trị truyền vào (args). Phương thức này không in ra màn hình (khác với System.out.printf()),
> mà chỉ trả về chuỗi kết quả để bạn có thể lưu vào biến hoặc in ra sau.
> ```java
> String name = "Alice";
> int age = 25;
> 
> String text = String.format("My name is %s and I am %d years old.", name, age);
> System.out.println(text);
> ```
> [!note]+ # StringBuilder và StringBuffer
> > [!note]+ ### Định nghĩa
> > > `StringBuilder` và `StringBuffer` dùng đề thay đổi một `String`
> > 
> > - Cho phép `thay đổi giá trị` nội dung mà `không tạo ra object mới`
> > - Có thể `add` `insert` `append`
> > - `StringBuffer` được đồng bộ, an toàn cho luồng (bảo toàn dữ liệu khi chạy đa luồng)
> > - `StringBuilder` chạy nhanh hơn, nhưng không an toàn khi chạy đa luồng
> > [!note]+ ### So sánh String/StringBuilder/StringBuffer
> > - `String` là `bất biến`, nếu ta cố thay đổi giá trị của nó, Java sẽ tạo ra `object mới`
> > - StringBuffer và StringBuilder là `khả biến` nên chúng có thể thay đổi giá trị
> >

> > [!note]+ ### Các phương thức
> > > [!note]+ String Buffer
> > > - public synchronized StringBuffer `**append**``(String s)` : Thêm chuỗi `s` vào cuối chuỗi hiện tại, có thể được `overload` để nhận các kiểu char, bool, int, float, double,…
> > > - public synchronized StringBuffer `**insert**``(int offset, String s)`: Chèn chuỗi `s` vào một vị trí `offset` trên chuỗi. Tương tự có thể được overload để nhận các giá trị như char, boolean, int, float, double
> > > - public synchronized StringBuffer `**replace**``(int start, int end, String s)` : Thay một chuỗi con bên trong chuỗi chỉ định bằng chuỗi `s`
> > > - public synchronized StringBuffer `**delete**``(int startIndex, int endIndex)` : Dùng để xoá một đoạn kí tự trong chuỗi.
> > > - public synchronized StringBuffer `**reverse**``()` : Lật ngược thứ tự các ký tự trong chuỗi
> > > - public int `capacity()`: Trả về sức chứa của Buffer hiện tại
> > > - public void `ensureCapacity(int minimumCapacity)`: Đảm bảo rằng sức chứa ở mức tối thiểu đưa vào
> > > - public char `charAt(int index)`: Trả về ký tự tại index
> > > - public int `length()`: Trả về độ dài của Buffer

> > > - public String `substring(int beginIndex, int endIndex)`:
